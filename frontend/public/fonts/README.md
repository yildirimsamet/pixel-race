# Fonts Directory

## MuybridgeGX.woff2

This directory should contain the MuybridgeGX.woff2 font file for animated horse emoji.

If the font file is not available, the system will fall back to:
- Apple Color Emoji (iOS/macOS)
- Segoe UI Emoji (Windows)
- Default system emoji font

The horse emoji 🐎 will be displayed using these fallback fonts if the custom font is not present.

To add the font:
1. Download or obtain MuybridgeGX.woff2
2. Place it in this directory
3. The font will be automatically loaded via globals.css
